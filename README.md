# Bollywood Movie Insights: Exploratory Data Analysis (EDA)

An exploratory data analysis project focused on understanding the financial performance, release timing, audience sentiment, and popularity metrics of Bollywood movies using Python.

##  Project Overview
This project analyzes a dataset of Bollywood films to uncover trends regarding:
- **Genre Distribution:** Identifying the most prominent film genres and their viewer demographics.
- **Financial Viability:** Evaluating Return on Investment (ROI) relative to production budgets and Box Office collections.
- **Release Strategy:** Determining the optimal times of the year (months/festive periods) for maximizing financial returns.
- **Social Media Engagement:** Analyzing user sentiment through YouTube views, likes, and dislikes, and mapping their correlation to box office success.

## Built With
- **Python 3**
- **Pandas & NumPy:** For data cleaning, manipulation, and feature engineering (such as calculating ROI and extracting month data).
- **Seaborn & Matplotlib:** For generating data visualizations, distribution plots, and correlation pair plots.
